# Security Policy

## Reporting a Vulnerability

If you've found a security issue in Quantum Cube, please **do not** open a public issue or pull request.

Report it privately via GitHub:

🔗 **[Report a vulnerability](https://github.com/quantumneurocreations-dot/quantumcube/security/advisories/new)**

The maintainer will be notified and can engage with you privately on a fix.

## Scope

**In scope:** [quantumcube.app](https://quantumcube.app), the Supabase Edge Functions backing it, and the source code in this repository.

**Out of scope:** vulnerabilities in third-party services we depend on (Supabase, Cloudflare, GitHub Pages, etc.) — please report those upstream. Automated-scanner findings without a working proof-of-concept are also out of scope.

## Response

This is a small project. Critical issues take priority. We aim to acknowledge reports promptly.

Thanks for helping keep Quantum Cube secure.
